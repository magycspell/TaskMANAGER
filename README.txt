
Task Manager(Python)

Este es un pequeño programa en Python que permite gestionar una lista de tareas desde una pequeña interfaz. Los usuarios pueden agregar tareas, marcarlas como completadas, ver la lista de tareas y eliminar las que ya no necesiten.
Funcionalidades

- Agregar una nueva tarea.
- Marcar tareas como completadas.
- Ver todas las tareas con su estado (pendiente o completada).
- Eliminar tareas de la lista.
- Salir del programa.

Cómo ejecutar

1. Asegúrate de tener Python instalado (versión 3 o superior).
2. Descarga el archivo del script (por ejemplo, `gestor_tareas.py`).
3. Ejecuta el programa desde la terminal con el siguiente comando:

bash
python Task_Manager.py
